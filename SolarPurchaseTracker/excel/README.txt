This folder is a suggested starting point for your "Database" folder.

When you open index.html and click "Open / Create Database Folder", you
can select this exact folder — the app will automatically create
Shipments.xlsx, Materials.xlsx, Vendors.xlsx, Items.xlsx and Settings.xlsx
inside it the first time you use the app.

If you already have an existing business workbook, you can also copy it
into this folder (or point the app at wherever it already lives) — the
app will detect it, keep every existing sheet untouched, and simply add
any of the five sheets above that are missing.
